Sterile Tool Transporter

a System for Sterilizing Surgical Tools using Remote Sensing and Actuation

The motivation behind programming an arm to transport the container to the sterilization autoclave was to reduce human effort and the sterilization would not be compromised if an arm can carry the container. The computation sub team was responsible for transferring a randomly spawned container and to the appropriate autoclave using muscle sensors. This project was coded with python on a raspberry pi

There are three distinct functions that correspond to three distinct muscle positions. When both arms are up, the arm moves. When only the right arm is up, the gripper opens or closes. When only the left arm is up the autoclave opens. The workflow is that it will start in home position, it will spawn a container, then it will move to the pickup location, then it will close the gripper, then it will move the arm to the autoclave bin, then it releases the container into the appropriate spot, if it is a large bin then the autoclave bin will open, then the gripper will release, and then it will return back to the home position.
